# 常用的git命令
## 查看本地代码与远程代码的区别
`git diff HEAD --`

## 丢弃本地上一次commit的本地缓存代码
`git checkout --`

## 查看提交到github的记录
`git log`

## 查看本地commit记录
`git reflog`

## 给最近的一次commit加上一个tag
`git tag '我是星期五提交的代码'`
